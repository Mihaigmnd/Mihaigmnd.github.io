🔍 How Data Flows to Mixpanel
  1. You click a button on this website
  2. JavaScript code runs and calls mixpanel.track()
  3. Event is sent to Mixpanel servers (https://api-eu.mixpanel.com)
  4. Data appears in your Mixpanel dashboard within seconds

All data is sent automatically - you don't need to do anything!

📊 View Your Data in Mixpanel
  1. Go to mixpanel.com
  2. Log in to your account
  3. Select your project
  4. Go to Events tab to see all events
  5. Go to Users tab to see user properties
  6. Go to Insights to create reports
💻 Code Examples

Track a simple event:

mixpanel.track('Button Clicked', {})

Track event with properties:

mixpanel.track('Purchase', { amount: 29.99, product: 'Laptop', currency: 'GBP' })

Identify a user:

mixpanel.identify('user_123')

Set user properties:

mixpanel.people.set({ plan: 'premium', email: 'user@example.com', signup_date: '2024-01-01' })

Increment a property:

mixpanel.people.increment('login_count', 1)

❓ FAQ

Q: Do I need to set up anything in Mixpanel?
A: No! Your token is already configured. Just use this website and data will flow automatically.

Q: How long until data appears in Mixpanel?
A: Usually within 1-5 seconds. Check the Events tab in your Mixpanel dashboard.

Q: Can I see events in real-time?
A: Yes! Go to Mixpanel → Live View to see events as they happen.

Q: What if events aren't showing up?
A: Check that your token is correct and you're logged into the right Mixpanel project.

Q: Can I track custom properties?
A: Yes! Use the "Custom Event" section to add any properties you want.
